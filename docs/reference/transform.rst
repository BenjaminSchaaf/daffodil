.. d:module:: daffodil.transform

daffodil.transform
##################

The :d:mod:`daffodil.transform` module provides various transformation functions
that can be performed on images. Transform functions differ from filters
(:d:mod:`daffodil.filter`) in that they can be performed in-place.
